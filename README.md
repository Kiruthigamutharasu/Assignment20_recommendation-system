# Movie Recommendation System

## Project Description
This project implements a Content-Based Movie Recommendation System using Machine Learning.

The system recommends similar movies based on movie overview text using TF-IDF vectorization and cosine similarity.

## Dataset
TMDB 5000 Movie Dataset (Kaggle)

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## Features
- Text preprocessing
- TF-IDF vectorization
- Cosine similarity
- Top-N movie recommendations
- Streamlit web interface
